Definisi, Rumus - Rumus, Contoh Soal dan Grafik Fungsi Irasional


Definisi Fungsi Irasional

Fungsi irasional merupakan fungsi yan gdomain nya terletak dibawah tanda akar atau suatu fungsi yang variabel bebasnya terdapat dibawah tanda akar.

Rumus - Rumus Fungsi Irasional

1. Bentuk Umum Fungsi Irasional

Bentuk umum fungsi Irasional adalah sebagai berikut :


.$f(x)=\sqrt{ax^2 + bx + c + d}$ Dengan syarat $ax^2 + bx + c + d \geq 0, y \geq 0, a, b, c, d \in \mathbb{R}$

Bentuk-betuk fungsi irasional

  1. bentuk $y = \sqrt{x},x\geq0$
  2. bentuk $y = \sqrt{ax+c}$
  3. bentuk $y = d+\sqrt{ax}+c$


2. Domain & Range

Bentuk umum fungsi Irasional adalah sebagai berikut :


Misalkan $f(x)=\sqrt{3x-2}$
Syarat : $g(x) \geq0$
Maka : $3x-2 \geq 0 \Rightarrow 3x\geq 2 \Rightarrow x\geq 2/3$

Maka domain fungsi f(x) adalah $[2/3,+\infty), \{x\geq 2/3\}$
Dan rangge fungsi f(x) adalah $[0,+\infty), \{x\geq 0\}$


Menggambar Grafik Fungsi Irasional


  • Syarat dibawah tanda akar x ≥ 0
  • kuadrat kan kedua ruas:

    • $y = \sqrt{x}$
    $(y)^2 = \sqrt(x)^2$
    $y^2 = x$

  • Persamaan diubah kedalam bentuk x = f(y)

    • $x = y^2$

  • Mengambil titik uji pada sumbu y, kemudian substitusikan ke persamaan

    • $x = y^2$
    $Jika : y = 0 \Rightarrow x = 0^2 \Rightarrow x = 0 $
    $ y = 1 \Rightarrow x = 1^2 \Rightarrow x = 1 $
    $ y = 2 \Rightarrow x = 2^2 \Rightarrow x = 4 $
    jadi, titik-titiknya adalah (0,0), (1,1), (4,2)
