Fungsi Linear

Materi

Fungsi linear adalah fungsi yang membentuk garis lurus ketika digambarkan dalam sebuah grafik. Bentuk umum dari fungsi linear adalah y = mx + c, di mana m adalah gradien atau kemiringan garis, dan c adalah titik potong dengan sumbu y.

Rumus

y = mx + c

Contoh Soal

Misalkan m = 2 dan c = 3. Fungsi linearnya adalah y = 2x + 3. Hitung nilai y ketika x = 4.

Penyelesaian: y = 2(4) + 3 = 8 + 3 = 11.

Perhitungan Titik Koordinat

Untuk menentukan titik potong dengan sumbu x (y=0):

0 = mx + c → x = -c/m

Untuk menentukan titik potong dengan sumbu y (x=0):

y = c
