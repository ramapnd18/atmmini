Definisi, Rumus - Rumus, Contoh Soal dan Grafik Fungsi Linear


Definisi Fungsi Linear

fungsi linear adalah relasi yang memasangkan setiap anggota yang ada di himpunan A dengan anggota lain di himpunan B. sederhana nya fungsi linear adalah suatu fungsi yang membentuk grafik secara garis lurus.

Rumus - Rumus Fungsi Linear

1. Persamaan Linear satu variabel

Bentuk umum persamaan linear satu variabel adalah sebagai berikut :


dimana:

  • y adalah variabel tergantung (nilai keluaran),
  • x variabel bebas (nilai masukan),
  • m adalah gradien atau kemiringan garis (yang menunjukkan tingkat perubahan y terhadap x ),
  • c adalah konstanta atau intercept (titik potong garis dengan sumbu y).

Contoh Soal

Misalkan m = 2 dan c = 3. Fungsi linearnya adalah y = 2x + 3. Hitung nilai y ketika x = 4. Penyelesaian: y = 2(4) + 3 = 8 + 3 = 11.

2. Persamaan Linear dua variabel

Bentuk umum persamaan linear dua variabel adalah sebagai berikut :


dimana:

  • a , b dan c dalah konstanta,
  • x dan y adalah variabel.

Penjelasan Komponen:

  • Koefisien a dan b: Menentukan kemiringan garis. Kombinasi a dan b menentukan arah garis di bidang koordinat.
  • Konstanta c: Menentukan posisi garis terhadap sumbu koordinat. Nilai c dapat diubah untuk menggeser posisi garis tanpa mengubah kemiringannya.

Bentuk Gradien-Intercep :

Jika kita ingin mencari kemiringan atau gradien garis (m) dan titik potong dengan sumbu y (ntercept), kita bisa mengubah persamaan ini ke bentuk y = mx + c


Menggambar Grafik Fungsi Linear

Untuk menggambar suatu persamaan garis (Fungsi Linear) kita hanya membutuhkan dua buah titik yaitu :

  1. Titik potong terhadap sumbu X
    • Untuk mencari titik potong terhadap sumbu X maka kita harus memasukan nilai Y = 0 kedalam persamaan garis
    • Kemudian kita mencari nilai X
  2. Titik potong terhadap sumbu Y
    • Untuk mencari titik potong terhadap sumbu Y maka kita harus memasukan nilai X = 0 kedalam persamaan garis
    • Kemudian kita mencari nilai Y
